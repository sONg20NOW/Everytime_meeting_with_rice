# Everytime_meeting_with_rice
2025년도 성균관대학교 졸업 작품

## 개요
1. 에브리타임에 있는 시간표를 이용하여 사용자의 공강 시간대를 계산하고,
2. 이를 활용하여 밥약을 희망하는 공강 시간을 보유한 유저 간 매칭. (혹은 매치 요청)

## Variation
* 매칭 시 추가 조건 설정 가능
* "단순히 매칭을 희망하는 유저들 간 매칭" 혹은 "매칭 신청 후 승낙" 시스템 설정.

위와 같은 variation을 주는 다양한 매칭 모드 설정 가능.

## 프로젝트 계획
프론트엔드와 백엔드 모두 구현이 필요함.<br>
따라서 프론트엔드 구현을 1학기 목표로 잡고, 백엔드 구현을 2학기 목표로 잡는다.
